# Editor/Markdown 目录结构

```
editor/
├── markdown/
│   ├── components/
│   │   ├── StatusBar.tsx          # 状态栏组件，显示字数统计等信息
│   │   ├── ToolbarButton.tsx      # 工具栏按钮组件
│   │   ├── markdownPreview.tsx    # Markdown 预览组件
│   │   └── showToolbar.tsx        # 工具栏显示组件
│   ├── markdownEditor.tsx         # 主编辑器组件
│   ├── type.ts                    # 类型定义文件
│   └── utils.ts                   # 工具函数
├── markdown功能说明.md             # Markdown 编辑器功能说明文档
├── sample.md                      # Markdown 示例文件
└── 使用示例.md                     # 使用示例文档
```

## 文件说明

### `markdownEditor.tsx`
- **功能**: 主要的 Markdown 编辑器组件
- **特性**: 
  - 支持编辑/预览/分栏三种模式
  - 实时字数和字符数统计
  - 工具栏支持常用 Markdown 语法
  - 同步滚动功能
  - 主题切换（亮色/暗色）

### `type.ts`
- **功能**: 定义 TypeScript 类型和接口
- **内容**:
  - `EditorMode` 类型（'edit' | 'preview' | 'split'）
  - `MarkdownEditorLayoutProps` 接口

### `utils.ts`
- **功能**: 包含工具函数
- **内容**:
  - `calculateWordCount` - 计算字数
  - `calculateCharCount` - 计算字符数
  - `getLanguageFromExtension` - 从文件扩展名获取编程语言

### `components/` 目录
- **`StatusBar.tsx`**: 显示编辑器状态信息（字数、字符数等）
- **`ToolbarButton.tsx`**: 可复用的工具栏按钮组件
- **`markdownPreview.tsx`**: Markdown 渲染预览组件
- **`showToolbar.tsx`**: 工具栏主组件，包含格式化按钮和模式切换

## 组件关系图

```
markdownEditor.tsx (主组件)
├── components/showToolbar.tsx (工具栏)
│   ├── components/ToolbarButton.tsx (工具按钮)
├── components/markdownPreview.tsx (预览组件)
├── components/StatusBar.tsx (状态栏)
├── type.ts (类型定义)
└── utils.ts (工具函数)
```

## 依赖关系

- 使用 `react-markdown` 进行 Markdown 渲染
- 使用 `remark-gfm` 支持 GitHub Flavored Markdown
- 使用 `rehype-highlight` 实现代码语法高亮
- 使用 `lucide-react` 提供图标
- 与 `@/lib/store/cdn/markdown/markdown_style` 集成实现样式管理
