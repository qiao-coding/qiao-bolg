# Markdown 编辑器使用指南

这是一个功能齐全的 Markdown 编辑器，支持实时预览和多种编辑模式。

## 功能特性

- **实时预览** - 编辑时即时查看渲染效果
- **多种模式** - 编辑模式、预览模式和分栏模式
- **语法高亮** - 代码块支持语法高亮
- **字数统计** - 实时显示字数和字符数
- **工具栏** - 快速访问常用 Markdown 语法

## 基本语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

```markdown
**粗体文字**
*斜体文字*
***粗斜体文字***
```

### 列表

#### 无序列表
```markdown
- 项目 1
- 项目 2
  - 子项目
```

#### 有序列表
```markdown
1. 第一项
2. 第二项
3. 第三项
```

### 代码

行内代码：`const code = "示例代码";`

代码块：
```javascript
function helloWorld() {
  console.log("Hello, world!");
}
```

### 引用

> 这是一个引用块
> 可以包含多行内容

## 使用技巧

1. 使用工具栏按钮快速插入格式
2. 在分栏模式下可以同时查看编辑和预览
3. 利用快捷键提高编辑效率
4. 代码块会自动检测语言并应用语法高亮

## 快捷操作

- **Ctrl+S** 保存内容
- **Ctrl+Z** 撤销操作
- **Ctrl+Y** 重做操作

这个编辑器支持 GitHub Flavored Markdown (GFM) 语法，可以满足日常写作的所有需求。
